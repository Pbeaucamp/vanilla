About vanilla

    BPM BIPortal is the entry door to the BPM BI's suit of platforms and latest technologies available such as FreeDashboard, FreeAnalysis, FreeWebReport and FreeMetrics. BPM BIPortal is based on GWT, a proven Web 2.0 Framework powered by Google Dev Team, and provides a powerful interface composed with tabulation and customizable folders. BIPortal adds Web2.0 features such as Single Sign On, easy access to documentation, easy-to-use interface to access both documents and Web Design Tools. For administrators, BIPortal provides functions such as User & Group management, user activity management, security and content management.

    BIPortal provides additional features such as schedule management (to periodically run reports and received result in either a Web folder or any personal mailbox), document management (access to history of previously available reports, search engine, type of document and key word management), possibility to enter notes and actions on each document, direct access to FreeMetrics Alerts, management of personal WatchList, creation of Workbook (assembly of different document in a consistent user package)
