//READ ME

-> Open Command Line
-> Go to the folder containing the jar
-> Type following command:

java -jar bpm.vanilla.package.build.jar [PATH_TO_PACKAGE] [PATH_TO_DEPLOY]

Example:

java -jar bpm.vanilla.package.build.jar C:/Users/Sébastien/Desktop/My_Samples.vanillapackage C:/User/Sébastien/Desktop

(Carefull no / at the end of the PATH_TO_DEPLOY)
